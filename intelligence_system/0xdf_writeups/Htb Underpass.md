I‚Äôll pull data from SNMP to find a daloRADIUS server on UnderPass. I‚Äôll find the login page, and use default creds to get access. There I‚Äôll find a hash for a user, which can be cracked to get SSH access to the box. That use can run a Mobile Shell (Mosh) server as root using sudo, and that leads to a root shell.

### dalorRADIUS

#### Background

There is a reference in the SNMP data about Underpass being a ‚Äúdaloradius‚Äù server. [daloRADIUS](https://www.daloradius.com/) is a UI management tool for Radius that boasts:

> No more mysql database console.
> No more editing freeradius configuration files.
>
> with daloRADIUS you can easily and quickly manage your FreeRADIUS deployment, thanks to flexible user interface and navigation flow.

Remote Authentication Dial-In User Service, or [RADIUS](https://en.wikipedia.org/wiki/RADIUS), is:

> a networking protocol that provides centralized authentication, authorization, and accounting ([AAA](https://en.wikipedia.org/wiki/AAA_(computer_security))) management for users who connect and use a network service.

#### Find Base Path

Without much else to go on, I‚Äôll try visiting `/daloradius`, and there is a different response:
